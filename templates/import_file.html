Импортировать файлы можно несколькими способами: перемещением необходимых файлов в папку Assets в корневой папке проекта или непосредственно в проект, переместив необходимые файлы в выделенную зону.

Так же в выделенной области можно создавать свои папки и прочее путём нажатия в её пределах ПКМ и выбрав необходимое действие.

Создадим несколько папок (Folder) и импортируем несколько спрайтов, которые можно найти на просторах интернета или нарисовать самому в любых пригодных для этого программах, для разработки игры.
